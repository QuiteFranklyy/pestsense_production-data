## Bar Graph Properties
| Property | Description |
| -------- | ----------- |
| Title | Sets a title |
| Chart Legend | Menu Item. |
| Chart Color | Set the color of the chart. | 
| Chart Color Hover | Sets the color when mouse hover is active on the line.
| Line Type | Type of line displayed on the graph. |
| History Timespan | History hours range to display. |
| Custom Domain | Adds a custom domain. |
